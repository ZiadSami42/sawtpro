# 🎙️ صوتبرو (SawtPro)

**صوتبرو** هو معرض أعمال احترافي للمؤدي الصوتي، يجمع بين الأناقة في التصميم والذكاء في الوظائف. يهدف المشروع إلى تقديم واجهة عرض عصرية للعينات الصوتية مع دمج أدوات ذكاء اصطناعي لمساعدة العملاء في تحسين نصوصهم قبل طلب الخدمة.

## ✨ المميزات

- **معرض أعمال تفاعلي:** استخدام `WaveSurfer.js` لعرض الموجات الصوتية بشكل ديناميكي بدلاً من المشغلات التقليدية.
- **مساعد النصوص الذكي:** أداة مدعومة بـ `Gemini AI` تقوم بتشكيل النصوص لغوياً واقتراح النبرة الصوتية الأنسب (وثائقي، إعلاني، تمثيلي).
- **تصميم عصري:** واجهة مستخدم متجاوبة بالكامل مبنية باستخدام `Tailwind CSS` مع تأثيرات بصرية جذابة (Glassmorphism).
- **أداء سريع:** بنية خلفية خفيفة باستخدام `FastAPI` لضمان سرعة الاستجابة وأمان مفاتيح API.

## 🛠️ التقنيات المستخدمة

### الواجهة الأمامية (Frontend)
- **HTML5 & CSS3**
- **Tailwind CSS** (للتنسيق والتصميم)
- **JavaScript (ES6+)**
- **WaveSurfer.js** (لتمثيل الموجات الصوتية)
- **Lucide Icons** (للأيقونات)

### الواجهة الخلفية (Backend)
- **Python 3.11+**
- **FastAPI** (إطار عمل سريع لبناء APIs)
- **Google GenAI SDK** (للتكامل مع نماذج Gemini)
- **Uvicorn** (خادم ASGI)
- **python-dotenv** (لإدارة متغيرات البيئة)

## 🚀 التشغيل المحلي

### 1. المتطلبات
تأكد من تثبيت `Python 3.11` أو أحدث.

### 2. إعداد البيئة
```bash
# الدخول إلى مجلد المشروع
cd sawtpro

# إنشاء بيئة افتراضية وتفعيلها
python -m venv .venv
source .venv/bin/activate  # على Linux/Mac
# .venv\Scripts\activate  # على Windows

# تثبيت الاعتمادات
pip install -r requirements.txt
```

### 3. إعداد مفتاح API
قم بإنشاء ملف `.env` في المجلد الرئيسي للمشروع وأضف مفتاح Gemini الخاص بك:
```env
GEMINI_API_KEY=your_api_key_here
```

### 4. تشغيل الخادم
```bash
uvicorn app.main:app --reload
```
سيكون الخادم متاحاً على `http://127.0.0.1:8000`.

### 5. تشغيل الواجهة الأمامية
يمكنك ببساطة فتح ملف `index.html` في أي متصفح حديث.

## 🌐 النشر (Deployment)

- **الواجهة الأمامية:** يتم نشرها عبر **GitHub Pages**.
- **الواجهة الخلفية:** يمكن نشرها على منصات مثل **Render**, **Railway**, أو **DigitalOcean**.
