# 🎙️ صوتبرو (SawtPro)

**صوتبرو** هو معرض أعمال احترافي للمؤدي الصوتي، يجمع بين الأناقة في التصميم والذكاء في الوظائف. يهدف المشروع إلى تقديم واجهة عرض عصرية للعينات الصوتية مع دمج أدوات ذكاء اصطناعي لمساعدة العملاء في تحسين نصوصهم قبل طلب الخدمة.

## ✨ المميزات

- **معرض أعمال تفاعلي:** استخدام `WaveSurfer.js` لعرض الموجات الصوتية بشكل ديناميكي بدلاً من المشغلات التقليدية، مع استضافة الملفات محلياً لضمان أقصى سرعة وأداء.
- **مساعد النصوص الذكي:** أداة مدعومة بـ `Gemini AI` تعمل مباشرة من المتصفح لتقوم بتشكيل النصوص لغوياً واقتراح النبرة الصوتية الأنسب (وثائقي، إعلاني، تمثيلي).
- **تصميم عصري:** واجهة مستخدم متجاوبة بالكامل مبنية باستخدام `Tailwind CSS` مع تأثيرات بصرية جذابة (Glassmorphism).
- **بنية ثابتة (Static):** الموقع مصمم ليعمل بكفاءة عالية على GitHub Pages دون الحاجة إلى خادم خلفي معقد.

## 🛠️ التقنيات المستخدمة

### الواجهة الأمامية (Frontend)
- **HTML5 & CSS3**
- **Tailwind CSS** (للتنسيق والتصميم)
- **JavaScript (ES6+)**
- **WaveSurfer.js** (لتمثيل الموجات الصوتية)
- **Lucide Icons** (للأيقونات)
- **Google Gemini API** (للمعالجة الذكية للنصوص)

## 🚀 التشغيل المحلي

بما أن المشروع عبارة عن موقع ثابت (Static Website)، يمكنك تشغيله بطريقتين:

### الطريقة الأولى: الفتح المباشر (للمعاينة السريعة)
يمكنك ببساطة فتح ملف `index.html` في أي متصفح حديث. 
*(ملاحظة: قد تمنع بعض المتصفحات تشغيل الموجات الصوتية بسبب سياسات CORS عند الفتح المباشر).*

### الطريقة الثانية: استخدام خادم محلي (للتجربة الكاملة)
لضمان عمل جميع الميزات (بما في ذلك الموجات الصوتية) بشكل صحيح، يفضل تشغيل خادم بسيط باستخدام Python:

```bash
# الدخول إلى مجلد المشروع
cd sawtpro

# تشغيل خادم HTTP بسيط
python3 -m http.server 8000
```
بعد ذلك، قم بزيارة الرابط التالي في متصفحك: `http://localhost:8000`

### 🔑 إعداد مفتاح API
لتفعيل مساعد النصوص الذكي، يجب إضافة مفتاح Gemini API الخاص بك داخل ملف `index.html`:
ابحث عن السطر: `const apiKey = "";` وقم بوضع مفتاحك بين العلامتين.

## 🌐 النشر (Deployment)

الموقع مهيأ للنشر التلقائي عبر **GitHub Pages**. يتم استخدام **GitHub Actions** لرفع التحديثات تلقائياً إلى فرع `gh-pages` عند كل عملية دفع (Push) للفرع الرئيسي.
